# Amazon Products Sales - Data Analysis

---

# Objective

This project analyzes Amazon products sales data using Excel and Power BI to identify pricing patterns, customer behavior, product demand, and promotional impact.

The main objectives of this project are:

- Analyze product demand and pricing patterns
- Understand customer ratings and review behavior
- Evaluate coupon and sponsorship impact
- Create interactive Power BI dashboards
- Generate business insights from Amazon product data

---

# Dataset Information

- Source: Kaggle
- Domain: Sales & E-Commerce
- Dataset Year: 2025
- Total Products: 42,000+
- Total Columns: 16

## Column Types

- String/Text Columns: 12
- Numeric Columns: 3
- Date/Time Columns: 1

# Data Cleaning and Preparation

The following preprocessing steps were performed:

- Converted text-based ratings into numeric values using Rating_Clean column
- Standardized demand indicators using Demand_Level column
- Categorized products into Budget, Mid-Range, and Premium price groups
- Classified review counts into Few Reviews, Moderate Reviews, and Highly Reviewed
- Standardized coupon categories
- Cleaned sponsorship information using Sponsored_Status column
- Handled missing values and inconsistent formats

# Interactive Filters

Interactive slicers were added for:

- Price Range
- Sponsored Status
- Demand Level

These filters allow dynamic analysis and better exploration of product trends.

---

# Key Insights

- Budget and Mid-Range products dominate the dataset
- Most products fall under Medium Demand category
- Highly Reviewed products form the largest share
- Average product rating is 4.39, indicating strong customer satisfaction
- Organic products significantly outnumber sponsored products
- Coupon usage is limited across products
- Higher-priced products do not necessarily receive higher ratings
